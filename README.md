# Smart Door Access System for UMaT

This repository contains a simple Python implementation of a secure door access system designed for UMaT admin staff using encapsulation.

## Requirements
- Public attribute: `s_name`
- Private attribute: `__access_code`
- Length validation for access codes using the `@property` decorator

## Features
- Safe read access to private variables via getters
- Code length validation (must be exactly 6 characters) via setters
- Simple text display method for staff information
